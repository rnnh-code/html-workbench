About these examples

Each artifact above was generated end-to-end from a single prompt during the bundled eval run. No manual editing — what you see is what the skill produced. They are intentionally varied to span the eight capability categories the skill recognizes.

Every file is fully self-contained: no external scripts, no CDN, no remote fonts. Open any one of them on a phone, in a print preview, or in offline mode. They all keep working.

The skill also declines to produce HTML when Markdown would be clearer — for short answers, code-only responses, simple comparisons, or when the user explicitly asks for Markdown. Calibration matters as much as production.
